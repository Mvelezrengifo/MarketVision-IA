📖 Introducción
MarketVision IA es una plataforma inteligente de análisis financiero en tiempo real.
Combina:

Análisis técnico automatizado

Ingestión de datos en tiempo real

Procesamiento serverless

Modelos de IA generativa

👉 A diferencia de un dashboard tradicional, MarketVision interpreta tendencias, evalúa riesgo y genera razonamiento contextual automatizado.

Panel de Indicadores Técnicos: RSI, SMA, Sentimiento, Confianza IA

Consulta Instantánea: análisis cacheado en S3, respuesta <200 ms

⚠️ Interpretación de Riesgo
LOW → mercado estable, baja volatilidad

HIGH → incertidumbre, volatilidad elevada, noticias contradictorias
